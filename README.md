# White Home - تطبيق إدارة الطلبات 🏠

تطبيق PWA لإدارة طلبات التنظيف - يعمل على الهاتف بدون متجر تطبيقات.

---

## 🚀 النشر على Vercel (10 دقائق)

### الخطوة 1 — رفع على GitHub
```bash
git init
git add .
git commit -m "White Home PWA"
git branch -M main
git remote add origin https://github.com/USERNAME/whitehome-orders.git
git push -u origin main
```

### الخطوة 2 — النشر على Vercel
1. افتح [vercel.com](https://vercel.com) وسجل دخول
2. اضغط **"Add New Project"**
3. اختر الـ repo من GitHub
4. الإعدادات تلقائية (Vite يُكتشف تلقائياً)
5. اضغط **Deploy** ✅

---

## 📱 تثبيت على الهاتف

### Android (Chrome)
1. افتح رابط Vercel في Chrome
2. اضغط **⋮** (القائمة)
3. اختر **"إضافة إلى الشاشة الرئيسية"**
4. اضغط إضافة ✅

### iPhone (Safari)
1. افتح الرابط في **Safari** (مش Chrome)
2. اضغط زر **المشاركة** 🔗
3. اختر **"إضافة إلى الشاشة الرئيسية"**
4. اضغط إضافة ✅

---

## 💻 تشغيل محلي للتطوير
